##二叉树创建



###存储：
-  根据序列号链式存储
-  根据指针存储


```
         A
      /    \
     B      E
    / \     /
    C  D   F
先序：ABCDEF    （XOO）
中序：CBDAEF     (OXO)
后序：CDBFEA      (OOX)
```
 
1. 先序加后序不能确定左右根的排列  所以推理不出来树的结构
 



#####中后推结构：
1. 步骤1   由于先序  确定根节点 A    由中序以及根节点确定 左子树BCD 右子书EF
            A
         /     \
       BCD     EF
2. 步骤2   由先BCD和中序的CBD确定
           树结构
            B                    E
          / \                    /
         C  D                 F
3. 步骤3  由先序EF和中序EF确定右子树结构
              E
             /
           F

#####中前推结构：

1. **步骤1找到根节点**   
 - 由于后序  确定 `根节点 A `   由中序以及根节点确定 `左子树BCD` + `右子树EF`
            A
         /     \
       BCD     EF
2. 步骤2  
 - 由后序`BCD`和中序的`CBD`确定
           树结构
            B                    E
          / \                    /
         C  D                 F
3. 步骤3  
 - 由先序EF和中序EF确定右子树结构
              E
             /
           F


`前序+后序`不可推结构：存在结构推理的歧义性
